# Chatty 💬

A real-time fullstack chat application built with the MERN stack and Socket.io.

## Features
- Real-time messaging
- JWT authentication
- Profile picture uploads
- Online user presence indicators
- Clean responsive UI

## Tech Stack
React, Node.js, Express, MongoDB, Socket.io, TailwindCSS, Cloudinary

## Live Demo
- Frontend: https://fullstack-chatapp-new.vercel.app
- Backend: https://fullstack-chatapp-production-f44b.up.railway.app

## Setup
1. Clone the repo
2. Add `.env` in `/backend` with your MongoDB, JWT, and Cloudinary credentials
3. `cd backend && npm install && npm run dev`
4. `cd frontend && npm install && npm run dev`
